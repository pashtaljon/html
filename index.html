CBI demo

Демо в скрипте cbi-demo.js, пример использования функции в консоли разработчика:

getBalanceOf('0xb3C4F3BaBD5511792009e948c3Bdc0c282D5b8a2', (err, res) => {console.log(res.toString())})

Важно знать, что MetaMask не внедряет свой web3 в код страницы, если она открыта как файл. Страница должна быть передана в браузер по http(s).

Для запуска этого демо можно использовать простой web сервер - http-server из npm.
